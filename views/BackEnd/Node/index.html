    node 详细安装与配置

    # Node.js 网址

    传送门 安装完成后 cmd 执行以下命令出现版本号则表示安装成功

    node -v  
npm -v
    1
    2

    # 配置

    # 指定缓存路径

    node 通过 npm 安装的模块默认存放在C盘，如果不想占用C盘的空间，我们可以设置缓存路径

    1. 新建目录D:\node_cache
    2. 在该目录下创建node_cache和node_global两个文件夹
    3. 设置缓存路径
    npm config set cache "D:\node_cache\node_cache"
npm config set prefix "D:\node_cache\node_global"
    1
    2

    # 添加环境变量

    自定义缓存路径要配置环境变量，不然使用全局安装的模块的时候系统将会无法识别。将上一步的缓存路径D:\node_cache\node_global添加至系统变量 path 中

    # cnpm

    cmd 安装后执行 cnpm -v 出现版本号则配置成功

    npm install -g cnpm --registry=https://registry.npm.taobao.org
